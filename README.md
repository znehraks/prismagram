# prismagram
instagram clone with Express + Prisma + React and ReactNative

## User Stories

-[X] Create account
-[] Request Secret
-[] Confirm Secret(Login) 
-[] Like / Unlike a photo
-[] Comment on a photo
-[] Search by user
-[] Search by location
-[] See user profile
-[] Follow / unfollow User
-[] See the full Photo
-[] Edit my profile
-[] Upload a photo
-[] edit the photo (delete)
-[] See the feed
